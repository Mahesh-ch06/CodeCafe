2.1: The if Statement – Making Decisions

Explanation

The if statement allows you to make a decision based on a condition. If the condition is true, a certain block of code will run.

Syntax

if (condition) {
    // Code to execute if the condition is true
}

Real-Life Example

Imagine deciding if you need an umbrella. You’ll take it only if it’s raining.

Code Example

public class IfExample {
    public static void main(String[] args) {
        int temperature = 30;

        if (temperature > 25) {
            System.out.println("It's a hot day!");
        }
    }
}

2.2: Adding else for More Options

Explanation

The else statement allows you to define what happens when the if condition is false.

Real-Life Example

If you want to decide what to wear based on the weather:

  • If it’s cold, you wear a jacket.
  • Otherwise, you wear a t-shirt.

Code Example

int temperature = 20;
if (temperature > 25) {
    System.out.println("It's a warm day.");
} else {
    System.out.println("It's a cool day.");
}

2.3: The else if Ladder for Multiple Conditions

Explanation

The else if ladder allows you to check multiple conditions in a sequence. Each else if checks a new condition if the previous ones are false.

Real-Life Example

Imagine you’re categorizing people by age:

  • Children: age < 13
  • Teens: 13 <= age < 18
  • Adults: age >= 18

Code Example

int age = 16;
if (age < 13) {
    System.out.println("Child");
} else if (age < 18) {
    System.out.println("Teen");
} else {
    System.out.println("Adult");
}
