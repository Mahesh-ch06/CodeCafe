1.1: Basic Structure of a Java Program

Explanation

In Java, every program consists of classes and methods:

  • Class: A class is like a blueprint or template for creating an object (just like a recipe is a template for making a dish).
  • Main Method: The main method is the entry point for the program; it's where the program starts running. Think of it as the "go button" of a machine.

Code Structure

Here’s the simplest structure of a Java program:

public class MyFirstProgram {  // This is the class
    public static void main(String[] args) {  // This is the main method
        System.out.println("Hello, Java!");  // This line prints a message
    }
}

Explanation of Each Part

  1. public class MyFirstProgram: This line defines a class named MyFirstProgram. You can think of this as defining a type of "machine" or "template."
  1. public static void main(String[] args): This is the main method, which acts like the machine’s start button.
  1. System.out.println("Hello, Java!");: This line tells the program to print a message on the screen.

Real-Life Example

Think of a Java program like a coffee machine:

  • Class (MyFirstProgram): The design of the coffee machine.
  • Main Method: The "Start" button on the machine.
  • Print Statement: The machine printing a message, like "Coffee is ready!" when it’s done.

1.2: Understanding
System.out.println

Explanation

System.out.println is used to display information on the screen. It’s the basic way to output text or results.

  • System: Refers to the system environment where the program runs.
  • out: A predefined output object in Java.
  • println: Means "print line." It displays text and moves to the next line.

Code Example

public class PrintExample {
    public static void main(String[] args) {
        System.out.println("Learning Java!");
        System.out.println("This is fun!");
    }
}

1.3: Declaring Variables – Storing Information

Explanation

Variables store data that can be used in your program. Think of them like labeled storage boxes for different values:

  • int: Used to store whole numbers (e.g., 5, 100).
  • String: Used to store text (e.g., "Hello", "Java").
  • double: Used to store decimal numbers (e.g., 5.5, 99.99).

Code Example

public class VariableExample {
    public static void main(String[] args) {
        int age = 18;  // Storing age as a whole number
        String name = "Mahesh";  // Storing name as text
        double height = 5.8;  // Storing height as a decimal number

        System.out.println("Name: " + name);
        System.out.println("Age: " + age);
        System.out.println("Height: " + height);
    }
}

1.4: Basic Operations with Variables

Explanation

Once you have variables, you can perform operations on them:

  • Addition (+): Adds numbers or joins text.
  • Subtraction (), Multiplication (), Division (/): Perform arithmetic with numbers.
  • Arithmetic Operators: Used for basic math (+, , , /, %).
  • Assignment Operators: Used to assign values (=, +=, =, etc.).

Code Example

public class OperationsExample {
    public static void main(String[] args) {
        int num1 = 10;
        int num2 = 5;

        // Addition
        int sum = num1 + num2;
        System.out.println("Sum: " + sum);

        // Subtraction
        int difference = num1 - num2;
        System.out.println("Difference: " + difference);

        // Multiplication
        int product = num1 * num2;
        System.out.println("Product: " + product);

        // Division
        int quotient = num1 / num2;
        System.out.println("Quotient: " + quotient);
    }
}

Real-Life Example

Imagine you’re keeping track of items in a store. You can use addition to add items, subtraction if you sell some, and division if you need to split items into groups.

Imagine arithmetic operators as simple calculations in daily life:

  • + could represent adding money to a savings account.
  • could represent withdrawing money.
  • and / could represent splitting or pooling resources among friends.
