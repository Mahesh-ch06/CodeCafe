Step 5: Loops in Java

5.1: for Loop – Repeating Code a Fixed Number of Times

Explanation

A for loop is ideal when we know in advance how many times we want to repeat a block of code. It consists of three parts in its declaration: initialization, condition, and increment/decrement.

Syntax

for (initialization; condition; increment/decrement) {
    // Code to be repeated
}

Code Example

public class ForLoopExample {
    public static void main(String[] args) {
        // Print numbers from 1 to 5
        for (int i = 1; i <= 5; i++) {
            System.out.println("Number: " + i);
        }
    }

Real-Life Example

Imagine counting down the floors in an elevator. If you're on the 5th floor and want to go down to the 1st floor, you know how many times to go down (5 times), just like a for loop.

5.2: while Loop – Repeat Code While a Condition is True

Explanation

A while loop is useful when we don’t know in advance how many times the loop should run. Instead, the loop continues as long as a given condition is true.

Syntax

while (condition) {
    // Code to be repeated
}

Code Example

public class WhileLoopExample {
    public static void main(String[] args) {
        int count = 1;

        // Print numbers from 1 to 5
        while (count <= 5) {
            System.out.println("Count: " + count);
            count++;
        }
    }
}

Real-Life Example

Consider reading a book until you reach the end. You might not know the exact number of pages remaining, but you continue reading while there are pages left.

5.3: do-while Loop – Execute Code At Least Once

Explanation

The do-while loop is similar to the while loop, but it guarantees that the code block will execute at least once, regardless of the condition. The condition is checked after the loop’s code has run.

Syntax

do {
    // Code to be repeated
} while (condition);

Code Example

public class DoWhileExample {
    public static void main(String[] args) {
        int count = 1;

        // Print numbers from 1 to 5
        do {
            System.out.println("Count: " + count);
            count++;
        } while (count <= 5);
    }
}

Real-Life Example

Imagine going to a restaurant and being asked to try a new dish. You have to try it at least once before deciding whether to order more based on your taste.

5.4: for-each Loop – Simplified Loop for Arrays and Collections

Explanation

The for-each loop, also known as the enhanced for loop, is designed specifically for iterating over arrays or collections. It simplifies the loop structure and avoids the need for an index variable.

Syntax

for (Type variable : array) {
    // Code using 'variable'
}

Code Example

public class ForEachExample {
    public static void main(String[] args) {
        int[] numbers = {1, 2, 3, 4, 5};

        // Print each number in the array
        for (int number : numbers) {
            System.out.println("Number: " + number);
        }
    }
}

Real-Life Example

Imagine going through a list of items in your shopping cart. With for-each, you’re simply accessing each item directly without worrying about indices.

Summary and Practice

Loops are incredibly powerful tools that let us control repetitive tasks in programs. Each loop type (for, while, do-while, and for-each) has specific use cases where it’s most effective.
