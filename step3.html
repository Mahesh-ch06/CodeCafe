Step 3: Looping with for, while, and do-while

3.1: The for Loop – Counting with Precision

Explanation

The for loop is great for situations where you know exactly how many times you want to repeat a task. It has three parts:

  1. Initialization: Sets up the starting value (e.g., int i = 0).
  1. Condition: Tells the loop how long to run (e.g., i < 10).
  1. Increment/Decrement: Adjusting the loop variable each time (e.g., i++).

Syntax


for (initialization; condition; increment/decrement) {
    // code to repeat
}

Code Example


public class ForLoopExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.println("Counting: " + i);
        }
    }
}

Real-Life Example

Think of doing 5 push-ups. You know the number (5), so you use a for loop to do each push-up until you reach 5.

3.2: The while Loop – Repeating with Conditions

Explanation

The while loop repeats a block of code as long as a condition is true. It’s useful when you don’t know beforehand how many times you need to loop, but you do know the condition.

Syntax


while (condition) {
    // code to repeat
}

Code Example


public class WhileLoopExample {
    public static void main(String[] args) {
        int i = 1;
        while (i <= 5) {
            System.out.println("Step " + i);
            i++;
        }
    }
}

Real-Life Example

Imagine filling a glass with water from a pitcher. You keep pouring until the glass is full. Here, the condition is “glass not full,” so you stop pouring when the condition is no longer true.

3.3: The do-while Loop – Guaranteed One-Time Execution

Explanation

The do-while loop is like the while loop, but it guarantees that the code runs at least once, even if the condition is false from the start. This is helpful for tasks that need to be performed once before the condition is checked.

Syntax


do {
    // code to repeat
} while (condition);

Code Example


public class DoWhileExample {
    public static void main(String[] args) {
        int i = 1;
        do {
            System.out.println("Number: " + i);
            i++;
        } while (i <= 5);
    }
}

Real-Life Example

Think of pressing a light switch. You press it at least once even if it’s already on or off, and then check to see if the light has changed.

3.4: Using break and continue in Loops

Explanation

  1. break: Exits the loop immediately, regardless of the loop condition.
  1. continue: Skips the current iteration and moves to the next one.

Code Example


public class BreakContinueExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            if (i == 5) {
                break; // Exit the loop when i is 5
            }
            System.out.println("Number: " + i);
        }

        System.out.println("Using continue:");

        for (int j = 1; j <= 10; j++) {
            if (j % 2 == 0) {
                continue; // Skip even numbers
            }
            System.out.println("Odd Number: " + j);
        }
    }
}

Real-Life Example

  • break: Think of a break like a fire alarm; you immediately stop and exit the building.
  • continue: Imagine grading papers but only looking at certain sections; you skip others (continue).

3.5: Nested Loops – Loops Inside Loops

Explanation

Nested loops are loops inside another loop. They are useful for tasks like working with multi-dimensional arrays or repeating actions multiple times within other repeated actions.

Code Example


public class NestedLoopExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 3; i++) {
            for (int j = 1; j <= 3; j++) {
                System.out.println("i = " + i + ", j = " + j);
            }
        }
    }
}

Real-Life Example

Imagine a restaurant with tables and chairs. You’re checking each chair (inner loop) at each table (outer loop) to make sure they're clean.
