Step 7: Arrays in Java

7.1: Introduction to Arrays – What They Are and Why We Use Them

Explanation

An array is a data structure that can store a fixed number of elements of the same data type. Instead of creating multiple variables for similar data, you can use an array to store them in one place.

Syntax

dataType[] arrayName = new dataType[size];
  • dataType: The type of data the array will store (e.g., int, String, etc.).
  • arrayName: The variable name for the array.
  • size: The number of elements the array can hold (its capacity).

Example

int[] numbers = new int[5]; // Array to hold 5 integers

Real-Life Example

Think of an array like a row of lockers in a gym, each identified by a number. If you have 5 lockers (like numbers[5]), each locker can store a different item but only of a specific type (e.g., all lockers storing shoes).

7.2: Initializing and Accessing Array Elements

Explanation

Array elements are accessed using their index, with indexing starting from 0. You can initialize arrays with default values or by assigning specific values directly.

Syntax for Initializing an Array with Values

int[] numbers = {1, 2, 3, 4, 5};

Code Example

public class ArrayExample {
    public static void main(String[] args) {
        int[] numbers = {10, 20, 30, 40, 50};

        // Access and print each element
        for (int i = 0; i < numbers.length; i++) {
            System.out.println("Element at index " + i + ": " + numbers[i]);
        }
    }
}

7.3: Modifying Array Elements

Explanation

You can change an element's value by accessing its index and assigning it a new value.

Code Example

public class ModifyArray {
    public static void main(String[] args) {
        int[] numbers = {1, 2, 3, 4, 5};

        // Change the value at index 2
        numbers[2] = 10;

        // Print the modified array
        for (int number : numbers) {
            System.out.println(number);
        }
    }
}

7.4: Multi-Dimensional Arrays

Explanation

Multi-dimensional arrays (often 2D arrays) are like arrays of arrays, useful for representing grid-like data structures like matrices or tables.

Syntax

dataType[][] arrayName = new dataType[rows][columns];

Code Example

public class MultiDimArray {
    public static void main(String[] args) {
        int[][] matrix = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };

        // Print each element in the 2D array
        for (int i = 0; i < matrix.length; i++) {
            for (int j = 0; j < matrix[i].length; j++) {
                System.out.print(matrix[i][j] + " ");
            }
            System.out.println();
        }
    }
}

Real-Life Example

A 2D array is like a grid of seats in a cinema hall. Each row and column combination represents a seat.

7.5: Array Length and Iteration

Explanation

The length property in Java provides the total number of elements in an array. You can use it to control loops when iterating through the array.

Code Example

public class ArrayLengthExample {
    public static void main(String[] args) {
        int[] scores = {85, 90, 78, 92, 88};

        System.out.println("Number of elements: " + scores.length);

        // Print each element
        for (int i = 0; i < scores.length; i++) {
            System.out.println("Score: " + scores[i]);
        }
    }
}
