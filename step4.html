4.1:if Statement – Basic Conditional Check

Explanation

The if statement allows the program to execute a block of code only if a specified condition is true.

Code Example

java
Copy code
public class IfStatementExample {
    public static void main(String[] args) {
        int temperature = 30;

        // Checking if the temperature is greater than 25
        if (temperature > 25) {
            System.out.println("It's a hot day!");
        }
    }
}

Real-Life Example

Think of the if statement like deciding to carry an umbrella if it’s raining. If it’s raining (condition is true), you take the umbrella. If it’s not, you don’t.

4.2: if-else Statement – Dual Conditional Check

Explanation

The if-else statement allows you to specify an alternative block of code to execute if the condition is false.

Code Example

java
Copy code
public class IfElseExample {
    public static void main(String[] args) {
        int age = 20;

        // Checking if the person is an adult
        if (age >= 18) {
            System.out.println("You are an adult.");
        } else {
            System.out.println("You are not an adult.");
        }
    }
}

Real-Life Example

Imagine checking if a movie is suitable for kids. If it’s rated “G,” kids can watch it; otherwise, they can’t.

4.3: Using && (AND) and || (OR) Operators

Explanation

Logical operators help combine multiple conditions:

  • && (AND): Both conditions need to be true.
  • || (OR): At least one of the conditions needs to be true.

Syntax

java
Copy code
if (condition1 && condition2) {
    // code to execute if both conditions are true
}
if (condition1 || condition2) {
    // code to execute if at least one condition is true
}

Code Example

java
Copy code
public class LogicalOperatorsExample {
    public static void main(String[] args) {
        int age = 20;
        boolean hasID = true;

        if (age >= 18 && hasID) {
            System.out.println("You are allowed to enter.");
        } else {
            System.out.println("Access denied.");
        }
    }
}

Real-Life Example

Imagine a movie theater where you need to be at least 18 and have a valid ID to watch a restricted movie. Both conditions need to be true to get access.

4.4: Combining Multiple if-else Statements with else if

Explanation

else if allows you to add more than one alternative to your if statements, creating a series of choices.

Code Example

java
Copy code
public class AgeGroupExample {
    public static void main(String[] args) {
        int age = 15;

        if (age < 13) {
            System.out.println("Child");
        } else if (age < 18) {
            System.out.println("Teenager");
        } else if (age < 65) {
            System.out.println("Adult");
        } else {
            System.out.println("Senior");
        }
    }
}

Real-Life Example

Consider categorizing people based on age:

  • If they are less than 13, they are a "Child."
  • If they are 13 to 17, they are a "Teenager."
  • If they are 18 to 64, they are an "Adult."
  • If they are 65 or older, they are a "Senior."

4.5: The switch Statement – Clean Multi-Choice Conditions

Explanation

The switch statement is useful when you need to test a variable against multiple values. It’s often simpler than using a long if-else if ladder.

Code Example

java
Copy code
public class SwitchExample {
    public static void main(String[] args) {
        int day = 3;

        switch (day) {
            case 1:
                System.out.println("Monday");
                break;
            case 2:
                System.out.println("Tuesday");
                break;
            case 3:
                System.out.println("Wednesday");
                break;
            case 4:
                System.out.println("Thursday");
                break;
            case 5:
                System.out.println("Friday");
                break;
            default:
                System.out.println("Weekend");
                break;
        }
    }
}

Real-Life Example

Think of a switch statement like selecting options on a vending machine: if you press “1” for chips, “2” for soda, etc., the machine delivers the specific item.

4.6: Ternary Operator – Compact Conditional Expressions

Explanation

The ternary operator (?:) is a shorthand for if-else that assigns a value based on a condition. It follows this format: condition ? value_if_true : value_if_false.

Code Example

java
Copy code
public class TernaryOperatorExample {
    public static void main(String[] args) {
        int age = 18;
        String result = (age >= 18) ? "Adult" : "Minor";
        System.out.println("You are an " + result);
    }
}

Real-Life Example

Imagine a traffic signal with a “Go” or “Stop” indicator based on whether the light is green or red. You could use the ternary operator to assign “Go” or “Stop” based on the color.
